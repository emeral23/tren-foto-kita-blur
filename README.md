## Installation

Clone this repository:

```bash
git clone 
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run
python blur.py

Press `ESC` to close the application.

## Tech Stack

* Python
* OpenCV
* MediaPipe
