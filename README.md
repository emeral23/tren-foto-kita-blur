## Installation

Clone this repository:

```bash
git clone https://github.com/emeral23/tren-foto-kita-blur.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run
python blur.py

Press `ESC` to close the application.

## Tech Stack

* Python
* OpenCV
* MediaPipe
